## [1.0.0] - 2024-03-19

### 新增
- 基础的画作评分系统
  - 支持图片上传和预览
  - 基于颜色丰富度和画面大小的评分
  - 根据评分提供鼓励性反馈
- 用户友好的 Web 界面
  - 支持拖拽上传
  - 实时图片预览
  - 响应式设计
- 完整的测试套件
  - 单元测试
  - 集成测试
  - 错误处理测试

### 技术细节
- 使用 Flask 框架构建后端
- 使用 Pillow 进行图像处理
- 采用 TDD 开发方式
- 实现错误处理和日志记录
